# @whirlpool/config

Loads the **whirlpool** user config file (TOML) and exposes a typed `WhirlpoolConfig` for cache clean paths and related settings.

## Main exports

- `loadConfig()` — read and parse config
- `WhirlpoolConfig` — configuration type

## Dependencies

- `smol-toml` — TOML parsing

## Scripts

`build`, `lint`, `check-types`.
